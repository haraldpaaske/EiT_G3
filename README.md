# EiT_G3
robotic arm sim

## Using python virtual environment
Linux:
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
Windows:
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## To save the requirements
Linux:
```bash
pip freeze > requirements.txt
```
